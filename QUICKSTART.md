```
  task_2_bfs.py      - Задание 2: Поиск в ширину (BFS)  
  task_3_dijkstra.py - Задание 3: Алгоритм Дейкстры

```

```

Доступные задания:
```

## Примеры входных данных

### Для заданий 1, 2, 4 (неориентированный граф):
```
a b
a c
b c
b d
c d
```
(затем нажмите Enter для завершения ввода)

### Для задания 3 (ориентированный взвешенный граф):
```
a b 5
a c 2
b d 3
c d 4
d e 1
```
(затем нажмите Enter для завершения ввода)

## Клавиши управления

- **Enter** - Завершить ввод графа или подтвердить выбор
- **Ctrl+C** - Завершить программу (из любого места)
- **Выбор 0** - Выход из программы (из главного меню)

```bash
python3 demo.py
```

Вы должны увидеть демонстрацию работы всех 4 заданий с примерами данных.

## Решение проблем

### Ошибка "ModuleNotFoundError: No module named 'tasks'"
- Убедитесь, что вы находитесь в папке проекта
- Проверьте, что папка `tasks/` существует и содержит файлы `.py`

### Программа не отвечает на ввод
- Убедитесь, что вводите корректный номер (0-4)
- Нажмите Enter после ввода

### Граф не добавляется
- Проверьте формат: должно быть ровно 2 элемента (для 1,2,4) или 3 (для 3)
- Элементы разделяются пробелом
- Пример правильного ввода: `a b` или `a b 5`
